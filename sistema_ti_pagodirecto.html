Configurar sincronización
Conectar con Google Sheets via Apps Script — una sola vez.
1
Subir el Google Sheets a Drive
Abre el archivo .xlsx incluido en el proyecto. Súbelo a Google Drive y conviértelo a Google Sheets.
2
Abrir Apps Script
En el Sheets: Extensiones → Apps Script. Pegar el código del archivo apps_script.gs del proyecto.
3
Publicar como Web App
Apps Script: Implementar → Nueva implementación → Aplicación web. Ejecutar como: yo. Acceso: Cualquier usuario.
Copiar la URL resultante: https://script.google.com/macros/s/[ID]/exec
4
Pegar la URL en Configuración
Ir a Config → campo URL del Web App → pegar → Guardar. ¡Listo! El sistema ya sincroniza automáticamente.
